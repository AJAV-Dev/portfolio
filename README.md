Este sitio fue construido con el objetivo de:

- 📄 Servir como CV interactivo

- 🌐 Estar disponible desde cualquier dispositivo (¡es completamente *responsive*!)

Puedes visitar el portafolio en: 👉 [https://tu-dominio-personal.com](https://tu-dominio-personal.com)

## 🛠️ Tecnologías y herramientas

- **Frontend:** React + TypeScript + Next.js
- **Estilos:** TailwindCSS
- **Componentes UI:** Shadcn/UI
- **Formulario de contacto:** Ruta API integrada con envío simulado
- **Control de versiones:** Git + GitHub
